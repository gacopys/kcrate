  <how-to-verify>
    Run from the repo root directory:

    Step 1 — Start the stack (first run may take 2-5 minutes for Maven to download dependencies):
      docker compose up
    Watch the logs in the terminal. You can also run `docker compose up -d` and then `docker compose logs -f`.

    Step 2 — Verify proxy-builder exited cleanly:
      docker compose ps proxy-builder
    Expected: Status shows `Exited (0)` — exit code 0 means the JAR was built and copied successfully.
    Failure: `Exited (1)` means Maven build failed — check `docker compose logs proxy-builder`.

    Step 3 — Verify CrateDB cluster formed (3 nodes):
      curl -s -X POST http://localhost:4200/_sql \
        -H "Content-Type: application/json" \
        -d '{"stmt":"SELECT count(*) FROM sys.nodes"}'
    Expected: {"cols":["count"],"rows":[[3]],...} — count must be 3.
    Acceptable for Phase 1: count of 1 means cluster is still forming; wait 30-60 seconds and retry.

    Step 4 — Verify no ClassNotFoundException in Keycloak logs:
      docker compose logs keycloak 2>&1 | grep -i "classnotfound"
    Expected: No output (empty — no ClassNotFoundException).
    Failure: Any line containing `ClassNotFoundException` means the proxy JAR was not loaded.

    Step 5 — Confirm Keycloak loaded the proxy driver:
      docker compose logs keycloak 2>&1 | grep -i "crateproxy\|crate-proxy\|provider"
    Look for any log line referencing the provider being loaded. Keycloak logs provider scanning at startup.

    Step 6 (optional) — Check Keycloak is up at port 8080:
      curl -s -o /dev/null -w "%{http_code}" http://localhost:8080/
    Note: Keycloak will likely show errors because BEGIN is not yet swallowed (Phase 2). EXPECTED per D-02.

    What is acceptable for Phase 1 (per D-02):
    - Keycloak crashing or entering a boot-loop after loading the driver IS ACCEPTABLE
    - Keycloak complaining about BEGIN or transaction errors IS ACCEPTABLE
    - ClassNotFoundException IS NOT acceptable — if this appears, the phase is not complete
  </how-to-verify>
